## Hospital Queuing System (Swing application)

### Build Requirement
- Maven
- JDK 21 (Tested)

### Run Requirement
- JVM 21 (Tested)

### Build
#### `mvn clean package` to build app

### Run
#### copy app.db to `/path/to` directory
#### `java -cp "/path/to/hospital-queuing-system-v*.*.*.jar" com.parintorn0.app.HospitalQueue` to run app
