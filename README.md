## Hospital Queuing System (Swing application)

### Build Requirement
- Maven
- JDK 21 (Tested)

### Run Requirement
- JVM 21 (Tested)

### Build and Run
`mvn clean package` to build app

`java -cp "target/hospital-queuing-system-1.0-SNAPSHOT.jar" com.parintorn0.app.HospitalQueue` to run app

### Run
`java -cp "target/hospital-queuing-system-1.0-SNAPSHOT.jar" com.parintorn0.app.HospitalQueue` to run app
